# 🏦 Дашборд мониторинга драгметаллов и курсов валют ЦБ РФ

Интерактивный веб-дашборд для отслеживания динамики цен на драгоценные металлы и курсов валют от Центрального Банка России в реальном времени.

## ✨ Возможности

- 📊 **Драгметаллы**: золото, серебро, платина, палладий (руб/грамм)
- 💱 **Валюты**: USD, EUR, CNY, GBP и другие (руб за единицу валюты)
- 📈 **Интерактивные графики**: дневные, недельные, месячные данные
- 📉 **Аналитика**: абсолютные и процентные изменения (1д, 1н, 1м)
- ⚙️ **Гибкие настройки**:
  - Нормирование графиков (100 в начале периода)
  - Логарифмическая шкала
  - Разная гранулярность (день/неделя/месяц)
- 🔄 **Кэширование данных**: оптимизация запросов к API ЦБ
- 📱 **Адаптивный интерфейс**: работает на ПК, планшетах, смартфонах
